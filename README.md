## Скрипт и шаблон для мониторинга УТМ

Шаблон создавался для Zabbix 7.0 LTS с целью отслеживания работы УТМ (Универсальный Транспортный Модуль) ЕГАИС на магазинах и своевременного оповещения о проблемах.

![Скриншот 1](https://raw.githubusercontent.com/mark-bernstorf/zabbix-utm-egais/refs/heads/main/screenshots/screen0.png)

### Получаемые параметры

- доступность сервиса
- версия модуля
- время создания базы
- дата и время самого старого чека в буфере
- время самого старого чека в буфере в минутах
- дата и время истечения срока действия ГОСТ / RSA сертификата
- дата истечения срока действия ГОСТ / RSA сертификата в днях
- дата и время выпуска сертификата ГОСТ / RSA
- дата выпуска сертификата ГОСТ / RSA в днях

### Настроенные триггеры

> **Примечание:** В макросах задаются основные настройки, в т.ч. время технологического окна (если служба или сервер перезагружается по расписанию)

### Установка

#### 1. УСТАНОВИТЬ СКРИПТ

1. Скопировать файл `UTMINFO` из папки `script` в папку `/usr/lib/zabbix/externalscripts/`
2. Сделать файл исполняемым

ИЛИ

```bash
git clone https://github.com/mark-bernstorf/zabbix-utm-egais.git
sudo cp ./zabbix-utm-egais/script/UTMINFO /usr/lib/zabbix/externalscripts/
sudo chmod +x /usr/lib/zabbix/externalscripts/UTMINFO
```

Можете проверить, что всё сделали правильно, выполнив в терминале:

`/usr/lib/zabbix/externalscripts/UTMINFO status http://192.168.1.1`

Скрипт ожидает два входных параметра - что проверяем (status) и адрес УТМ (http://192.168.1.1) 

В ответ получим либо 0, если всё плохо по этому адресу, либо 1, если всё хорошо.

Вместо status можно указать и другой аргумент, список:

- status - доступность службы УТМ
- version - версия
- db_time - время создания БД
- rsa_release - дата выпуска RSA сертификата
- rsa_release_days - сколько дней назад выпущен RSA сертификат
- rsa_expiry - дата истечения RSA сертификата
- rsa_expiry_days - дни до истечения RSA сертификата
- gost_release - дата выпуска ГОСТ сертификата
- gost_release_days - сколько дней назад выпущен ГОСТ сертификат
- gost_expiry - дата истечения ГОСТ сертификата
- gost_expiry_days - дни до истечения ГОСТ сертификата
- buffer - дата самого старого неотправленного чека в буфере
- buffer_minutes - сколько минут висит самый старый неотправленный чек в буфере
- buffer_days - сколько дней висит самый старый неотправленный чек в буфере

#### 2. НАСТРОИТЬ ЗАБИКС

В Zabbix импортировать шаблон zabbix-utm-egais-template.yaml из папки template и подключить его к нужным узлам сети.
